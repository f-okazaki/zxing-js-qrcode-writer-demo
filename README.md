# ZXing

## Project in Maintenance Mode Only でしたので、独自に拡張してみました

ライブラリは、typescriptでしたので、Buildしてzxing.jsにすべてまとめています。


## usage
dockerで起動するようにしましたが、
html/css/jsのみで動作するため、index.htmlをブラウザで開くだけでも動作します。

## 動作に必要なファイル
index.html
lib.js
zxing.js

```
docker-compose up -d
```
